# Plano da Fase 2

## Objetivo

Transformar o protótipo inicial de lógica em uma base com cara de jogo idle/tycoon mobile.

## Loop principal

1. As estações trabalham automaticamente.
2. A barra de progresso enche.
3. O serviço conclui.
4. O dinheiro entra no caixa.
5. Um popup de dinheiro aparece.
6. O jogador compra upgrades.
7. A renda por segundo aumenta.
8. Novas estações são liberadas.
9. O prestígio reinicia a loja com multiplicador permanente.

## Elementos visuais adicionados

- Cabeçalho com dinheiro, renda e reputação.
- Oficina visual com clientes, balcão, bancadas e caixa.
- Clientes animados passando pela oficina.
- Lista de estações com barra de progresso.
- Botões grandes para ações principais.
- Toasts para feedback rápido.
- Modal de ganho offline.
