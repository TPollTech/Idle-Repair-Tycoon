# Idle Assistência - Fase 2

Esta fase transforma o protótipo inicial em uma versão com cara mais próxima de jogo idle mobile.

## Mudanças implementadas

- Layout vertical 1080x1920 pensado para celular.
- Oficina visual com fluxo: clientes -> balcão -> bancadas -> caixa.
- Clientes animados andando pela tela.
- Estações com progresso visual.
- Popups de dinheiro quando a grana entra.
- Barra inferior com Turbo 2x, pacote de dinheiro e prestígio.
- Renda offline com opção normal ou dobrada por anúncio fake.
- Save local via PlayerPrefs.
- Sem dependência de assets externos.

## Como testar

1. Abra o projeto no Unity.
2. Abra qualquer cena vazia.
3. Clique em Play.
4. Na aba Game, use 9:16 ou 1080x1920 Portrait.
5. Ajuste o Scale para 0.5x ou 1x se a tela parecer cortada.

## Observação

Os anúncios ainda são fake para teste. A integração real com AdMob fica para a próxima etapa de monetização.
